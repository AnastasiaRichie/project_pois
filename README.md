# Планирование дежурств по парковкам

Веб-приложение для менеджеров и сотрудников: зоны на карте, назначение смен, учёт машин и отчёты. Регистрация с подтверждением менеджером; вход по почте/паролю или через Google.

**Что внутри.** Менеджер чертит и редактирует парковочные зоны (карта, Leaflet), назначает людей на даты — события уходят в Google Calendar. Сотрудник видит свои дежурства, данные по машинам собираются через Google Form в таблицу, из неё можно подтянуть записи в систему. Отчёты по зоне и периоду собираются в Google Docs (статистика по длительности стоянки).

**Стек.** Фронт — React 18 и Vite (`client/`). Бэкенд — Express, JWT, SQLite через better-sqlite3 (`server/`). С Google связаны OAuth, Calendar, Sheets, Docs/Drive.

**Запуск локально.** В `server/` — `npm install`, настроить `.env`, при необходимости `npm run migrate`, потом `npm run dev`. В `client/` — `npm install` и `npm run dev`.
